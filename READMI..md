# Запуск проекта

- Создаём виртуальное окружение:
python -m venv python-django

- Заходим в виртуальное окружение и активируем его:
cd Scripts
activate

- устанавливаем в виртуальное окружение Django и обновляем pip:
pip install django

- разворачием проект на Django:
django-admin startproject python_django
